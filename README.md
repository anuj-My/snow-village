# Noida's Snow Village — Frontend Assignment

### 🎯 Key Highlights

- **Figma to Code:** Pixel-perfect translation of layouts, color palettes, and typography.
- **100% Responsive:** Seamless adaptive design across mobile, tablet, and desktop devices.
- **GSAP Animations:** Premium scroll-triggered animations powered by GSAP & ScrollTrigger.

### 🛠️ Tech Stack

- **Framework:** Next.js (App Router, React 19)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** GSAP & ScrollTrigger

### 🚀 Quick Start

- **Install dependencies:** `npm install`
- **Run local server:** `npm run dev`
